Pré-requisitos

Esse curso vai prover para você todo o conhecimento necessário para você criar seus príprios aplicativos usando Ionic 2, usando muito hands-on e com uma abordagem focada em mostrar exemplos do mundo real. Nós iremos criar 4 aplicativos completos, além de diversos exemplos de boas práticas e técnicas interessantes pra você usar no seu dia-a-dia de desenvolvedor mobile.

Quais são os requisitos?
  • Notebook rodando Windows, Mac, ou Linux (Mac é necessário para build de apps iOS)
  • Conhecimento razoável de HTML, CSS, e JavaScript
  • Conhecimento básico de alguma linguagem orientada a Objetos
  • Desejável: conhecimento prévio de Angular, mas não é essencial
  • Desejável: experiência com linha de comando
